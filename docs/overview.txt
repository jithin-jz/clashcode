CLASHCODE - OVERVIEW
====================

What is this?
-------------
CLASHCODE is a gamified coding platform where you can solve programming challenges, earn XP, and level up your skills. It's built for developers who want a fun, interactive way to practice coding.

Main Features:
--------------
- Gamified Learning: Unlock challenges, earn stars, and collect badges.
- Real-time Interaction: Chat with other developers while you code.
- AI Tutor: Get hints and code reviews powered by AI (using RAG).
- Secure Execution: Your code runs in a safe, isolated container.
- Marketplace: Spend your hard-earned XP on cosmetic items.

How it works:
-------------
The platform is split into several microservices:
- Core: Handles the users, challenges, and the XP economy.
- Chat: Manages real-time messages and user presence.
- AI: Provides intelligent hints and code analysis.
- Executor: Securely runs user-submitted code.
- Analytics: Tracks progress and platform metrics.

Technology:
-----------
We use a modern stack including React 19 for the frontend, Django and FastAPI for the backend, and various databases like PostgreSQL, Redis, and DynamoDB.
