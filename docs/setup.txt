CLASHCODE - SETUP GUIDE
=======================

Prerequisites:
--------------
You'll need Docker, Node.js, Python 3.11+, and Git.

Local Development:
------------------
1. Clone the repo:
   git clone https://github.com/jithin-jz/clashcode.git

2. Environment Files:
   You need to copy the .env.example files to .env in these folders:
   - services/
   - services/core/
   - services/chat/
   - services/ai/
   - frontend/

3. Start the Backend:
   cd services
   docker compose up -d --build

4. Database Setup:
   docker compose exec core python manage.py migrate
   docker compose exec core python manage.py createsuperuser
   docker compose exec core python manage.py load_challenges

5. Start the Frontend:
   cd frontend
   npm install
   npm run dev

Common Commands:
----------------
- Stop everything: docker compose down
- Check logs: docker compose logs -f
- Database shell: docker compose exec db psql -U clashcode

Troubleshooting:
----------------
- If port 80 is busy, change NGINX_HTTP_PORT in services/.env.
- Check if the 'db' container is running if you can't log in.
